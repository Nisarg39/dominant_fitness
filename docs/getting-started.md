# Getting Started

## Prerequisites
- Node.js 18+
- npm or yarn or pnpm or bun

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```
Visit http://localhost:3000

## Build
```bash
npm run build && npm start
```

## Project Layout (key paths)
- src/app: App Router pages/layout
- src/components: UI components
- src/server: server actions, models, db config
- tailwind.config.ts: Tailwind 4 setup
- next.config.ts: images, headers, performance
